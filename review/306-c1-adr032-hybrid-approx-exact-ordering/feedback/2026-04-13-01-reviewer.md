# Feedback: C1 ADR-032 Hybrid Approx-Exact Ordering

**Reviewer:** Claude (Opus 4.6)
**Date:** 2026-04-13

## Verdict: Open packet, awaiting measurement — but likely facing the same ceiling

The motivation is correct: packet 305 showed that full exact replacement
of approximate ordering actively hurts. So a hybrid policy that preserves
some approximate signal is the logical next experiment.

However, I'm skeptical this will break through the 0.8080 ceiling for a
structural reason: the problem is not the *ordering policy within the
frontier* — it's the *graph neighborhoods the search visits*. At low ef,
the search only has budget to expand ~40 sources. If the approximate
scorer sends those expansions into a different part of the graph than the
exact scorer would, no ordering policy within the discovered set can
recover the undiscovered candidates.

### Recommendation

Run the measurement. If recall stays at 0.8080, this confirms the
structural hypothesis and the ADR-032 low-ef quality recovery effort
should pivot away from frontier-ordering strategies entirely.

If recall improves, it would mean the frontier ordering *does* affect
which sources get expanded (via the priority queue), and there's signal
in the hybrid approach worth pursuing further.
