# Review: SPIRE Strict Epoch Tracker Closure (30955)

## Summary

Tracker-only Phase 12.7 row closure citing 30895 Stage E
`epoch_mismatch` strict artifact:

- two dispatches: one ready remote + one stale-epoch remote;
- `expected_status=stale_epoch`;
- `expected_blocked_before_dispatch_count=1`;
- `expected_degraded_skipped_dispatch_count=0`;
- `expected_next_executor_step=remote_epoch_window`.

Observed line matches expectations → strict mode does NOT
mix the ready remote with the incompatible-epoch remote.

## Findings

### P3 — Citation sufficiency for strict epoch-mixing row only

The 30895 artifact is exactly the right evidence for the
strict epoch-mixing row:

- It exercises the joint case (ready + incompatible epoch),
  not just the single-remote stale case.
- The `expected_blocked_before_dispatch_count=1` +
  `expected_degraded_skipped_dispatch_count=0` pair is the
  load-bearing assertion: in strict mode, the incompatible
  remote blocks BEFORE dispatch (counted in blocked) and is
  NOT degraded-skipped (count zero).
- `expected_next_executor_step=remote_epoch_window` confirms
  the executor surfaces the *reason* for the block, giving
  an operator a remediation hook.

That's sufficient to close the strict epoch-mixing row. The
fixture proves the no-mix invariant.

### P3 — Tracker wording — still-open Phase 12.7 rows

The closure correctly does NOT claim:

- multi-instance setup;
- placement metadata;
- boundary-replica;
- Stage E rerun rows.

Those remain open. As long as the tracker says the strict
epoch-mixing row is closed by the cited 30895 artifact and
nothing more, the closure is honest. Per the request wording
this is the case.

### P3 — Degraded-mode counterpart

The strict closure cites the strict artifact only. If the
Phase 12.7 lane also tracks a degraded epoch_mismatch row
(degraded mode degrades-skips the incompatible remote and
returns a partial answer with the ready one), that row
should cite a parallel degraded artifact or remain explicitly
open. The 30954 degraded skip report would be the natural
observability surface for that path. Worth confirming the
tracker isn't accidentally bundling the degraded case under
this strict closure.

## Reviewer recommendation

Accept. Tracker-only closure with sufficient cited evidence
for the named row. The 30895 strict artifact is the right
proof of the no-mix invariant.
