## Per-axis findings (condensed)

Full per-axis detail in the audit conversation; this packet keeps the
condensed citations so the 12c task file can be reviewed against the
underlying evidence.

- **Axis A — CustomScan FFI lifecycle.** `recheck` always-true return
  unpinned; rescan-after-exhaustion has no live SQL fixture;
  End-after-cancel palloc/pfree pairing untested; `MarkPos`/`RestrPos`
  planner-exclusion untested; two `#[should_panic]` scaffolds at
  `custom_scan.rs:777,838` will need inversion.
- **Axis B — Transport happy path.** All CustomScan loopback fixtures
  are 1-remote; no parametric N (N=8/N=64) test;
  `transport_faults.rs:2` asserts overlap not parallelism; no fanout=3
  CustomScan fixture; selected-PID round-trip not asserted against
  returned-row PIDs.
- **Axis C — Transport failure modes.** See table above. New 12a
  categories and Stage E lifecycle rows have no live coverage.
- **Axis D — DML frontdoor.** Concurrent DELETE collision absent;
  composite-PK / float-PK / numeric-out-of-int8 PK rejection absent;
  DELETE-idempotent test asserts only row counts, not the
  `accepted/deleted_count=0` contract from packet `30980`; non-PK
  SELECT pass-through tested only at hook-installation level.
- **Axis E — Concurrency.** Long-scan + DROP INDEX (coord-side) absent;
  long-scan + remote restart absent; idle-in-transaction timeout
  absent; coordinator-crash-mid-2PC `prepare_acked` in-doubt window
  absent.
- **Axis F — Isolation.** `catalog_cleanup_policy.rs:839` matrix is
  missing `READ COMMITTED` row; EvalPlanQual / recheck-true contract
  unpinned.
- **Axis G — Planner / cost.** Cost-GUC override EXPLAIN reflection
  absent; empty-placement planner-refusal positive test absent (only
  the eligibility-false negative is tested).
- **Axis H — Multi-store width.** Two-store covered; three-store /
  four-store absent.
- **Axis I — Recall.** Only `30980`'s single-query spot check;
  `nprobe` sweep absent; no CI-runnable SPIRE recall pin.
- **Axis J — Data-shape edge cases.** Single-row corpus, all-duplicate
  vectors, numerical extremes, NUL-byte/control-char text, ≥1 MB
  strings, wide projection — all absent.
- **Axis K — Operator surface.** Stage E matrix tests are
  contract-only (assert row exists, not that executor performs the
  prescribed action); diagnostic snapshot survival under DROP
  INDEX/REINDEX absent.

## Semantic concerns (10 weakest existing tests)

1. `custom_scan.rs:46` — JSON-EXPLAIN substring asserts; no
   returned-PID equals selected-PID; only 2 rows inserted.
2. `custom_scan.rs:777,838` — `#[should_panic(expected=...)]`
   scaffolds; prove the planner+begin reach a placeholder, not
   positive behaviour.
3. `tuple_heap.rs:202` — asserts NULL array round-trip but not
   wire-byte shape; encoder regression writing zero-length instead of
   the NULL sentinel would slip past.
4. `production_summary.rs:234,290` — Stage E matrix tests are
   string-equality contract tests; do not exercise executor.
5. `custom_scan/tests.rs:351-403` — cost monotonicity uses loose `>`;
   sign-flip regression slipping past `high > low` is possible if
   the constant term dominates.
6. `catalog_cleanup_policy.rs:729,839-840` — isolation matrix missing
   `READ COMMITTED`.
7. `dml_frontdoor.rs:2323` — DELETE idempotent test asserts only row
   counts, not the documented response shape.
8. `insert.rs:993` — descriptor race asserts second insert succeeds;
   doesn't pin which descriptor generation won, doesn't assert zero
   orphan placements.
9. `custom_scan.rs:288` — empty-result test pins JSON status strings
   but not `EndCustomScan` invocation count or palloc/pfree pairing.
10. `coordinator/tests.rs:115` — sign convention pin is one 2-dim
    case (`score == -11.0`); no high-dim, NaN, dimension-mismatch.
