- **Result column count `9` and template are duplicated between
  this packet and 30568's `remote_search_libpq_result_contract_rows`.**

  builder returns a `Vec` of length 9 in 30568. Drift risk: a v2
  endpoint that adds a column would need to update both
  representations. Suggest deriving the column count from
  `remote_search_libpq_result_contract_rows().len()` rather than a
  literal. Cross-cutting summary on 30568.
