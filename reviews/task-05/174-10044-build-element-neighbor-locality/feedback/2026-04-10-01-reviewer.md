## Findings

### 1. The landed rewrite matches the request and preserves page-layout safety

Commit `3c57be4` fixes the bulk-build locality issue in the narrowest safe way:

- phase 1 writes a placeholder neighbor tuple followed immediately by the element tuple
- the element stores its final `neighbortid` immediately
- phase 2 updates the placeholder neighbor tuple in place once all element TIDs are known

Because the placeholder and final neighbor payloads have the same slot width, the in-place
`update_neighbor(...)` path keeps the tuple length stable instead of reopening the older
element-backfill loop.
